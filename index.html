📖 Comment utiliser l'application

1

Préparer la liste des joueurs (fichier CSV)

Créez un fichier .csv avec un joueur par ligne, au format :

Nom,Prénom
Dupont,Jean
Martin,Sophie

Le séparateur peut être une virgule ou un point-virgule. Téléchargez le fichier exemple via le bouton « ⬇️ Exemple CSV » pour partir d'une base prête à l'emploi.

2

Configurer le tournoi

  • Type de concours : Doublette (2 joueurs/équipe) ou Triplette (3 joueurs/équipe).
  • Nombre de parties : le nombre de rondes à jouer (ex. 5).
  • Numéro du premier terrain : numérotation de départ des terrains.
  • Jouer 1 terrain sur 2 : cochez cette case pour sauter un terrain entre chaque match (terrains numérotés en impair ou en pair).
3

Importer les joueurs

Cliquez sur le champ Fichier CSV des joueurs et sélectionnez votre fichier. La liste des joueurs inscrits s'affiche aussitôt pour vérification.

4

Démarrer le tournoi

Cliquez sur 🚀 Démarrer le Tournoi. Le tirage au sort est effectué automatiquement pour chaque partie en limitant au maximum les répétitions de partenaires et d'adversaires.

5

Saisir les scores & résultats

Pour chaque match, cliquez sur l'équipe gagnante et renseignez les scores. Deux vues sont disponibles :

  • 📋 Vue détaillée : affiche les noms complets et les champs de saisie des scores.
  • 📊 Vue condensée : vue compacte affichant initiales + nom et scores en lecture rapide.
6

Consulter le classement

L'onglet 🏆 Classement affiche le classement final trié par : Victoires → Goal Average (points marqués moins points encaissés) → ordre alphabétique en cas d'égalité parfaite.

💾

Sauvegarde automatique

Toutes les données (joueurs, parties, scores) sont sauvegardées automatiquement dans le navigateur (localStorage). Vous pouvez fermer la page et retrouver votre tournoi à la prochaine ouverture.

Le bouton 🆕 Nouveau Concours permet de repartir de zéro (après confirmation).
